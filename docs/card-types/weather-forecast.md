# Weather Forecast

Weather Forecast is now part of the [Weather](/card-types/weather) card.

To show tomorrow's high and low temperature, create a **Weather** card and set **Display** to **Tomorrow**.

Older configurations that used the separate **Weather Forecast** card continue to load as a Weather card set to **Tomorrow**.
